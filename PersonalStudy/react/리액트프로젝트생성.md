# 프로젝트 생성 명령 설치
```js
// node 설치
https://nodejs.org/en/download/

// 리액트 프로젝트를 생성하는 create-react-app 설치
npm install -g create-react-app

// 리엑트 프로젝트 설치하기
// create-react-app [프로젝트명]
create-react-app tutorial
```
* 생생된 프로젝트 폴더로 이동
```js
cd tutorial
yarn start
```
# 프로젝트에 JSX 추가하기
